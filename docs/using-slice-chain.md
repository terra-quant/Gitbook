# Using XHAVIC

***

### MetaMask Wallet <a href="#other-wallets" id="other-wallets"></a>

Xhavic can be added as a custom network to any EVM-compatible wallet (i.e. [MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn), Coinbase wallets).

&#x20; Adding Xhavic as a custom network to MetaMask Wallet:

1. Open the MetaMask browser extension.
2. Open the network selection dropdown menu by clicking the dropdown button at the top of the extension.
3. Click the **Add network** button.
4. Click **Add a network manually**.
5.  In the **Add a network manually** dialog that appears, enter the following information for the Xhavic testnet:

    | Name            | Value                                                 |
    | --------------- | ----------------------------------------------------- |
    | Network Name    | Xhavic Test Network                                   |
    | Description     | The public testnet for Xhavic.                        |
    | RPC Endpoint    | [https](using-slice-chain.md)://testrpc.xhaviscan.com |
    | Chain ID        | 16585                                                 |
    | Currency Symbol | XVC                                                   |
    | Block Explorer  | [https://](using-slice-chain.md)xhaviscan.com         |
6. Tap the Save button to save Xhavic as a network.

You should now be able to connect to the Xhavic by selecting it from the network selection dropdown menu.
